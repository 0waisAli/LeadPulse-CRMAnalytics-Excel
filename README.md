# LeadPulse - CRM Analytics - Excel

## Overview

A complete CRM analytics workflow built entirely in Excel, simulating a real-world marketing operations environment. This project takes messy, raw CRM data — the kind you'd export from HubSpot or Salesforce — and transforms it into a scored, segmented lead database with an executive dashboard.

The goal: demonstrate that Excel is a full analytics tool, not just a spreadsheet.

## Project Structure

| Sheet | Purpose |
|-------|---------|
| `raw_leads` | 1,000 messy lead records — original data, untouched |
| `raw_spend` | 135 campaign spend records (10 campaigns × 12 months + duplicates) |
| `raw_interactions` | 3,000 touchpoint-level event logs |
| `clean_leads` | Formula-cleaned lead data (TRIM, UPPER, PROPER, IF, OR) |
| `clean_spend` | Standardized campaign data with calculated ROI metrics |
| `clean_interactions` | Normalized event log with engagement classifications |
| `model` | Merged dataset — leads enriched with interaction aggregates and spend metrics |
| `scoring_weights` | Configurable scoring parameters (adjustable inputs) |
| `segment_thresholds` | Configurable HOT/WARM/COLD cutoff values |
| `scoring_results` | Lead scores with segment classification and recommended actions |
| `KPIs` | Helper data tables and pivot tables powering the dashboard |
| `dashboard` | Final visual layer — KPI cards, 7 charts, top 10 leaderboard |

## Data Cleaning

### Raw Data Issues (Intentional)

The raw datasets contain every common data quality issue found in real CRM exports:

**raw_leads (1,000 rows, 19 columns)**
- Inconsistent company names: google, Google, GOOGLE, "  Google ", google inc
- Mixed casing across all text fields
- Leading/trailing whitespace
- ~8% missing emails, ~10% blank numeric fields
- demo_requested stored as: yes/YES/y/Y/true/TRUE/1/no/n/false/0/blank
- Missing statuses, industries, job titles, countries
- Phone numbers in 7 different formats
- Countries as full names, abbreviations, and mixed case (United States/USA/us)
- Revenue as inconsistent text ranges ($1M-10M vs 1M-10M vs Series A)
- ~6% blank dates

**raw_spend (135 rows, 12 columns)**
- 15 duplicate rows (same campaign + month, different name casing)
- Campaign names in 38 variants for 10 campaigns
- Source and channel labels inconsistent

**raw_interactions (3,000 rows, 12 columns)**
- Interaction types in 4+ formats each (email_open/Email_Open/EMAIL_OPEN/email open)
- Device types needing grouping (iOS, Android → MOBILE)
- ~15% blank time_spent values
- Scattered sales rep notes with inconsistent casing

### Cleaning Approach

All cleaning was performed using Excel formulas — no VBA, no Power Query, no external tools.

| Transformation | Functions Used | Example |
|---------------|---------------|---------|
| Text standardization | TRIM, UPPER, PROPER, LOWER | "  Google " → "GOOGLE" |
| Null handling | IF, OR, ISNUMBER | blank → 0 (numerics), "N/A" (text) |
| Boolean mapping | IF, OR, UPPER, TRIM | "yes"/"Y"/"true"/"1" → "YES" |
| Variant consolidation | IF, OR, UPPER | "ecommerce"/"E-commerce" → "E-Commerce" |
| Country standardization | IF, OR, UPPER | "United States"/"USA"/"us" → "US" |
| Source normalization | IF, OR, UPPER, SUBSTITUTE | "cold email"/"Cold_Email" → "COLD_EMAIL" |
| Device grouping | IF, OR, UPPER | iOS/Android → "MOBILE" |
| Date handling | IF (pass-through) | Blanks preserved, dates unchanged |
| Data quality scoring | COUNTBLANK, IF | ≥8 blanks = LOW, ≥4 = MEDIUM, else HIGH |
| Deduplication | Remove Duplicates (Data tab) | 15 duplicate spend rows removed |

## Data Model

The `model` sheet joins all three cleaned datasets into a single enriched view per lead:

- **Lead profile** from clean_leads (company, size, source, industry, country)
- **Engagement metrics** aggregated from clean_interactions using COUNTIF/COUNTIFS/SUMIF:
  - Total interactions, high-engagement interactions, total time spent
  - Demo requests, content downloads, webinars attended, pricing page views
- **Source economics** from clean_spend using SUMIF:
  - Total spend on the lead's acquisition source
  - Total leads generated by that source
  - Cost per lead and ROAS for the source

## Lead Scoring System

### Scoring Weights (Configurable)

| Metric | Weight | Max Raw Value | Logic |
|--------|--------|--------------|-------|
| Email Opens | 10 | 35 | (opens / max) × weight |
| Clicks | 15 | 20 | (clicks / max) × weight |
| Website Visits | 10 | 60 | (visits / max) × weight |
| Demo Requested | 20 | 1 | YES = full weight, NO = 0 |
| Company Size | 15 | 1 | Enterprise=100%, Mid-Market=75%, SMB=50%, Startup=25% |
| Engagement Score | 30 | 1 | Composite of interactions, high engagement, demos, webinars, pricing views |

### Segmentation

| Segment | Score Threshold | Count | Action |
|---------|----------------|-------|--------|
| HOT | ≥ 60 | 110 (11%) | Schedule Call |
| WARM | ≥ 35 | 537 (54%) | Nurture Email Drip |
| COLD | < 35 | 353 (35%) | Retarget with Ads |

All weights and thresholds are stored in separate configuration sheets (`scoring_weights`, `segment_thresholds`) and referenced dynamically. Changing any value automatically recalculates all 1,000 lead scores.

## Dashboard

The dashboard includes:

- **8 KPI cards**: Total Leads, Hot/Warm/Cold counts, Avg Score, Total Spend, Avg CPL, Avg ROAS
- **Leads by Source**: Horizontal bar chart showing lead distribution across acquisition channels
- **Segment Distribution**: Donut chart — HOT (11%), WARM (54%), COLD (35%)
- **Monthly Spend Trend**: Line chart showing spend trajectory Jan–Dec 2025
- **Score Distribution**: Histogram of lead score buckets
- **Leads by Industry**: Bar chart across all industry verticals
- **Campaign Performance**: Clustered bar chart — spend vs revenue by campaign
- **Top 10 Leads**: Ranked leaderboard of highest-scoring leads

All charts are powered by pivot tables on the KPIs sheet for easy maintenance.

## Key Findings

1. **Organic and Webinar channels deliver the most leads** while maintaining lower cost per lead compared to paid channels
2. **54% of leads sit in the WARM segment** — the largest opportunity for nurture campaigns to convert them to HOT
3. **Only 11% of leads are HOT** — these should be prioritized for immediate sales outreach
4. **Campaign spend shows a clear Q4 ramp** — consistent with typical B2B budget cycles
5. **SEO Content Program has the highest ROAS** — strong argument for increasing organic investment

## How to Use

1. Download the `.xlsx` file
2. Open in Excel (Office 365 recommended for full formula support)
3. Start with the `dashboard` sheet for the overview
4. Explore `scoring_weights` and `segment_thresholds` to adjust parameters — all scores recalculate automatically
5. Review `clean_leads`, `clean_spend`, `clean_interactions` to see the formula-based cleaning
6. Check the `model` sheet to see how datasets are joined
